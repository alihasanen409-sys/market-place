# User Guide

## Buyers

- Browse listings from the marketplace page.
- Search by keyword.
- Open product details.
- Add a product to the cart.
- Use checkout for simulated payment.
- Open chat to message a seller.
- Save listings in favorites.
- Track orders in order history.

## Sellers

- Register with a seller role.
- Create a seller profile.
- Add listings with descriptions, pricing, tags, and category.
- Upload listing images and digital files through the backend API.
- Respond to buyers in conversations.
- Track sales from the seller dashboard.

## Admins

- Use Django admin at `/admin/`.
- Manage users, categories, listings, orders, reviews, and notifications.
- Approve or reject seller profiles.
- Moderate marketplace content.
